##### Given a number \***\*n\*\*** , the task is to return the count of digits in this number.

**Example:**

> **_Input:_** n = 1567
> **_Output:_** 4
> **_Explanation:_** There are 4 digits in 1567, which are 1, 5, 6 and 7.
>
> **_Input_**: n = 255
> **_Output_**: 3
> **_Explanation_**: There are 3 digits in 256, which are 2, 5 and 5.
